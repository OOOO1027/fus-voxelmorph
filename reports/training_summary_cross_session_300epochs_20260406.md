## 7. 关键发现

### 7.1 成功之处

1. **超越基线**：DL 模型在所有标准测试对上显著超越 Stage 3B
2. **泛化能力**：Test 集提升 +0.213，证明良好的跨 session 泛化
3. **物理合理**：Jacobian 分析确认变形场无折叠
4. **稳定收敛**：300 epochs 稳定收敛，无过拟合

### 7.2 局限性

1. **极端困难样本**：对于 Pre NCC = 0 的 session 对 (如 S2↔S3, S2↔S5)，DL 输出为 0
   - 可能原因：这些 session 间无有效 overlap，或预处理阶段失败
   - 建议：检查这些 session 的原始数据质量

2. **S7↔S16 提升有限**：Train 集中 S7→S16 仅提升 +0.001
   - 可能原因：Stage 3B 在此对上已表现很好 (NCC=0.934)
   - 剩余变形可能主要是噪声
